    • RCC Confirmation Widget

    Confirmation widgets are placed on the partners confirmation pages for their products.

    The partner will need to fill the values of the iframe with their customers information to see the relevant results. This can be done via either;

  • Longitude & Latitude (A set of 2 coordinates)
    • eg: "&latitude=48.8566&longitude=2.3522" Paris France
  • IATA code (A 3 letter Airport data code)
    • eg: "&CDG" Paris Charles de Gaulle Airport

    The design is fully responsive so sizing isnâ€™t too much of a problem. By default the widgets is styled with our RCC brand, this offers a co-brand logo with the option to hide. For white label partners the widget is available to be styled on a more in-depth basis in terms of color and brand.

    All widgets can be customised by number the of results shown & click through destination. Each touch-point is tracked so we know exactly how the customer interacted with the widget.

    As the widget will placed on a confirmation page it's importnat to keep this a secue link, either using our secure.rentalcars.com or your own ssl secure sub-domain

    Widget Features

    The RCC confirmtation widget is AB tested, allowing us to add in any new features once returned positive.

    • (NEW) An Improved speed and loading time
    • (NEW) A supplier logo will now appear next to each car image
    • Search results that can be pulled out either via IATA code or Long lat.
    • IP detection which will automatically set the pricing structure for the country the user is in just as on the site
    • Customisable styling allowing the widget to match the clients branding requirements.(depending on the size of the affiliate)
    • Options to change the language, date, area and currency.
    • A responsive design, this allows the widget to fit to any size including mobile devices.

    Widget Customisation

    To customise the widget you will be able to change dates, times, languages etc.. by inputting certain paramiters into the widget url.

    Affiliate Code

    Specifies to the widget the correct affiliate

    &affiliateCode=company

    Language

    The language to which to widget will display

    &preflang=en

    Pickup location

    The 3 letter IATA code which specifies the destination

    &pickupIATACode=MAN

    Pickup year

    Specify the pickup year

    &pickupYear=2016

    Pickup Hour

    Specify the pickup hour

    &pickupHour=12

    Return Hour

    Specify the return hour

    &returnHour=10

    Number of results

    Specify how many results you want to be displayed

    &results=3

    Sub-domain

    Specify a sub-domain for the search results to appear via

    &affUrl=carrentals.reservations.com

    Currency preference

    The currency in which you want to display

    &prefcurrency=GBP

    Pickup date

    Specify the pickup date

    &pickupDate=6

    Pickup month

    Specify the pickup month

    &pickupMonth=10

    Return date

    Specify the return date

    &returnDate=14

    Pickup Minute

    Specify the pickup Minute

    &pickupMinute=12

    Return Minute

    Specify the return minute

    &returnMinute=10

    Display price

    displays correct dates and not the average over 7 days

    &dateJump=false
