    • RCC Emerging Layer Banner

    The Dynamic Pricing banner was originally designed to be served in a popup window within the booking process however displaying it as a static banner is perfectly acceptable.

    The Dynamic pricing banner main functionality is to display a daily car rental price based on what our customers have found within the last 2 weeks. This is a cached price that is pulled in from our server.

    An airport IATA code can be feed in via a parameter in the URL, the name of the airport will be displayed in a sales message. We recommend displaying the pricing banner on the confirmation page of a flight or hotels booking process so that the IATA code can be dynamically feed in. This will insure the displayed car rental pricing is relevant to customer and increase conversion rates.

    Emerging Layer Banner Customisation

    To customise the ELB you will be able to change dates, times, languages etc.. by inputting certain paramiters into the banners url.

    Affiliate Code

    Specifies to the widget the correct affiliate

    &affiliateCode=company

    Language

    The language to which to widget will display

    &preflang=en

    Pickup location

    The 3 letter IATA code which specifies the destination

    &iataCode=MAN

    Pickup year

    Specify the pickup year

    &puYear=2016

    Pickup Hour

    Specify the pickup hour

    &puHour=12

    Return Hour

    Specify the return hour

    &doHour=10

    Background Image

    The display parameter is used to set the background image of the banner.

    &display=airport

    Sub-domain

    Specify a sub-domain for the search results to appear via

    &forwardUrl=carrentals.reservations.com

    Currency preference

    The currency in which you want to display

    &prefcurrency=GBP

    Pickup date

    Specify the pickup date

    &puDay=6

    Pickup month

    Specify the pickup month

    &puMonth=10

    Return date

    Specify the return date

    &doDay=14

    Pickup Minute

    Specify the pickup Minute

    &puMinute=12

    Return Minute

    Specify the return minute

    &doMinute=10

    Secure Domain

    This attribute is only required if the banner is being served via the secure domain

    &serverName=www.rentalcars.com
