    • Dynamic Location Banner

    This integration has been made to allow the customer be quickly taken to the Rental Cars homepage with preset destination information.

    The 'Dynamic Location Banner' supplies the user with a selection of countries along with the ability to specify pick up dates and drop off dates. After submission they will be directed to the rentalcars.com homepage where this criteria will be preselected allowing faster progression to the search results page.

    The banner is dynamic enabling you to enter city, country and location data via parameters ino the url. The style is customisable in terms of colour which can also be added in as a set parameter.

    Dynamic Location Banner Features

    The location banner can be customized in a varant of ways, including language, color, size and location.

    • Supports 42 languages
    • Country specific destinations
    • City specific destinations
    • Pre-select dropdown option
    • User defined button colour
    • User defined background colour
    • Supports all major browsers (including IE7)
    • Responsive width to fit the iframe
    • Minimal size (No jQuery)
    • Very detailed documentation

    Dynamic Location Banner Customisation

    To customise the location banner you will be able to change dates, times, languages etc.. by inputting certain paramiters into the banners url.

    Affiliate Code

    Specifies to the widget the correct affiliate

    &affiliateCode=company

    Language

    The language to which to widget will display

    &preflang=en

    Button Colour

    Use of a prefdefined colour may be used to stylise the content to your brand.

    &button=iris

    Hide RC Logo

    Hiding the rental cars logo

    &hideLogo=true

    Currency preference

    The currency in which you want to display

    &prefcurrency=GBP

    Background Colours

    Use of a prefdefined colour may be used to stylise the content to your brand.

    &background=blue

    Pre-set Size

    Specify a pre-seleted to size to work from

    w160h600 w180h140 w215h255 w250h160 w250h250 w300h200 w300h250 w300h300 w468h60 w470h160 w535h240 w728h250 w728h90 w750h190
